A Simple Light Box Plugin by Ashish Panchal.

Different options available in this plugin:

  1. Div from same page
  2. Internal Video
  3. YouTube video
  4. Page from Same/External domain
  5. Div from another page

How to use it?

HTML Code:

<ul>

<li><a data-target="internal" class="overlay" href="#example1">Div from same page</a></li>

<li><a data-target="internal" class="overlay1" href=".example2">Internal Video</a></li>

<li><a data-target="extvdo" class="overlay2" href="http://www.youtube.com/embed/WoAjE1aioHE" title="YouTube video">YouTube video</a></li>

<li><a data-target="external" class="overlay3" href="vdo-for-everyone.html" target="_blank">Page from Same/External domain</a></li>

<li><a data-target="extdiv" class="overlay" href="bxSlider.html#mover" target="_blank">Div from another page</a></li>

</ul>

JS Code:

<script src="siteware/js/jquery-1.7.1.js"></script>

<script src="siteware/js/jquery.aplightbox-v2.js"></script>

<script>

$(".overlay").aLightBox();

$(".overlay1").aLightBox({animation:true, direction:'left'});

$(".overlay2").aLightBox({title:true, titlePos:'top', display:'fade'});

$(".overlay3").apLightBox({animation:true, direction:'right'});

</script>

Features:

  • Important: You can import specific DIV from another page.
  • Can work with images, video, html pages.
  • Working on all most all browsers.
  • Accessible - you page would work if js is disabled.
  • Animation - from all four direction
  • fade or show

Defaults & Options

var defaults = {

animation:false, // DEFAULT = false, OPTION = true & false

direction:'top', // IF ANIMATION = true, DEFAULT = top, OTHER OPTIONS = top, bottom, left & right

time:700, // IF ANIMATION = true, DEFAULT = 700, YOU CAN SEN TIME IN MILISECONDS

display:'show', // DEFAULT = show, OPTION = fade & show

duration:500, // IF DISPLAY = fade, YOU CAN PUT DURATION IN MILISECONDS, DEFAULT = 500

top:'', // DEFAULT = center, OPTION = postion from top in pixel

left:'', // DEFAULT = center, OPTION = postion from left in pixel

title:false, // DEFAULT = false, OPTION = true & false

titlePos:'bottom', // IF TITLE = true, YOU CAN SET POSITION OF TITLE, DEFAULT = bottom, OPTIONS = top & bottom

customClass:'' // DEFAULT = blank, YOU CAN SET ONE OR MORE CLASS FOR SPECIFIC CSS

}

Note: I'm working on AJAX request for data for now (I'm working on it...) and rounded corners will not work on IE 7 & 8.
